Data Understanding & CleaningMemeriksa data kosong (Hasil: 0 missing values).Menghapus kolom non-numerik dan tidak relevan seperti id dan date.Menangani Outliers menggunakan metode Interquartile Range (IQR) pada kolom price dan sqft_living.Data awal: 21.613 baris.Data bersih: 20.467 baris.2.
